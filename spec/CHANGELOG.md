# Changelog

All notable changes to the ORS spec will be documented in this file.

## [v0.1] - 2026-02-26

### Added
- Initial schema spec for software engineer resumes
- Sections: basics, skills (primary/secondary), experience with structured impact, projects, learning
- Field definitions: depth enum, experience type enum, project role enum
- 14 validation rules
- JSON Schema for programmatic validation
- Design document with 9 key decisions and rationale
- Examples: staff-engineer (full) and minimal
